# Instructions # 
Comme d'habitude pour faire du sale avec ce repo en local 
```
git clone https://github.com/clydeat/vaxt
cd vaxt
bundle install 
rails db:create
rails db:migrate
rails s
```
Ensuite rends-toi sur :
<a href="http://localhost:3000">Localhost:3000</a>
